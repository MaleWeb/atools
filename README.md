## 轻松上手

```bash
# npm
npm install atools-js

# yarn
yarn add atools-js

# pnpm 🔥
pnpm add atools-js
```

> 如果你不想在项目中引入太多依赖，而又想使用某一个或几个方法
>
> 那么可以复制文档中的源码，在你的项目中引入

## 源码获取

```bash
# clone
git clone git@github.com:wangdaoo/atools.git

# enter
cd atools

# install
pnpm/npm/yarn install
```
